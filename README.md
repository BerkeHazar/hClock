# hClock

**hClock** is a simple, fast, and elegant clock application designed with a focus on aesthetics and performance.

## Features / Özellikler

### English
- **Liquid Glass UI**: Modern frosted glass effect with dynamic backgrounds.
- **Modes**:
  - **Clock**: Server-synced time for accuracy.
  - **Stopwatch**: Precise measurement with lap support (simple).
  - **Timer**: Quick presets (5m, 10m, etc.).
- **Themes**: Dark and Light mode support.
- **Internationalization**: Support for TR, EN, FR, DE.
- **Dynamic Backgrounds**: Beautiful curated nature photos or gradients.

### Türkçe
- **Sıvı Cam Arayüzü**: Dinamik arkaplanlar ile modern buzlu cam efekti.
- **Modlar**:
  - **Saat**: Kesinlik için sunucu senkronizasyonlu zaman.
  - **Kronometre**: Hassas ölçüm.
  - **Sayaç**: Hızlı ayarlar (5dk, 10dk vb.).
- **Temalar**: Koyu ve Açık mod desteği.
- **Dil Desteği**: TR, EN, FR, DE.
- **Dinamik Arkaplanlar**: Seçilmiş doğa fotoğrafları veya gradyanlar.

## Installation / Kurulum

```bash
git clone https://github.com/berkehazar/hClock.git
cd hClock
npm install
npm run dev
```
